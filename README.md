# AI-POWERED-SIGN-LANGUAGE-INTERPRETER

## Description

The AI Powered Sign Language Interpreter is designed to help bridge the communication gap between hearing-impaired individuals and others by translating sign language gestures into readable text or speech. The system uses Artificial Intelligence, Computer Vision, and Machine Learning techniques to recognize hand gestures in real time through a webcam.

## Technologies Used

* Python
* Machine Learning
* Artificial Intelligence
* OpenCV
* MediaPipe
* TensorFlow / Keras
* Flask (if used)

## Features

* Detects sign language gestures in real time
* Converts hand gestures into text output
* Supports communication for hearing-impaired users
* Fast and accurate gesture recognition
* User-friendly interface
* Can be extended for voice output

## How to Run the Project

1. Install Python on your system
2. Download or clone this repository
3. Install required libraries using: `pip install -r requirements.txt`
4. Run the main file: `python app.py`
5. Allow webcam access for gesture detection
